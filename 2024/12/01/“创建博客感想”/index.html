第一次创建博客

博客地址：https://github.com/xixi-1997/xixi.github.io

步骤：

安装nodejs
sudo pacman -S nodejs npm
查看版本并检验
node -V
npm -V
安装cnpm(国内源加速)
sudo npm install -g cnpm –registry=https://registry.npm.taobao.org
检查cnpm
cnpm
安装hexo
sudo cnpm install -g hexo-cli
检查hexo安装
hexo -v
初始化hexo
mkdir blog
cd blog
sudo hexo init 卡进度就考虑翻墙或者换源
启动博客
hexo s 本地默认地址localhost:4000
创建博客
hexo n “你的博客标题” 在source/_posts/生成md文件
清理与生成
hexo clean
hexo g
安装git部署插件
sudo cnpm install –save hexo-deployer-git
config设置
vim _config.yml 找到deployment
（在vim里面输入以下内容）
type: git
repo: git@github.com:yourname/yourname.github.io.git
branch: main
生成密匙
ssh-keygen -t rsa （会生成一个密匙信息，vim +密匙信息，会进入vim界面的密匙，再复制到GitHub中 设置。）
github提交密匙(SSH里设置)
检查是否密匙配置成功
ssh -T git@github.com
deploy到github
hexo d
完成

备注：cd 文件名 表示进入XX文件夹
hexo d 表示同步上传到GitHub云端
hexo n “XXX” 表示创建新的md格式文本博客
hexo s 表示进入本地端博客
